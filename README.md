# Stone-Paper-Chizzer

A compact Rock–Paper–Scissors–style game using Stone, Paper, and Chizzer. Play against the computer with simple rules, score tracking, and a minimal codebase ideal for learning or quick fun.

## Features
- Player vs computer gameplay
- Score tracking across rounds
- Lightweight, beginner-friendly implementation

## How to play
1. Choose one: Stone, Paper, or Chizzer.  
2. The computer chooses randomly.  
3. Rules:
   - Stone beats Chizzer
   - Chizzer beats Paper
   - Paper beats Stone
